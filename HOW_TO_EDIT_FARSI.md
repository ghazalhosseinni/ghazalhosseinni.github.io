# 🎯 چطوری متن‌های فارسی رو ویرایش کنم؟

## راه ساده (پیشنهادی) ✨

### قدم 1️⃣: فایل راهنما رو باز کن
```
📁 FARSI_TEXTS.md
```
این فایل **تمام متن‌های فارسی** سایت رو به صورت **خوانا و مرتب** داره.

### قدم 2️⃣: متن مورد نظرت رو پیدا کن
مثلاً اگه می‌خوای متن صفحه اصلی رو تغییر بدی، برو به بخش:
```
## 🎯 بخش 2: صفحه اصلی (Hero Section)
```

### قدم 3️⃣: متن جدید رو بنویس
فقط متن رو تایپ کن، نگران فرمت نباش!

### قدم 4️⃣: به فایل JSON برو
```
📁 src/i18n/locales/fa.json
```

### قدم 5️⃣: متن رو در JSON جایگزین کن
مثال:
```json
"hero": {
  "greeting": "متن جدید اینجا"  ← متن قدیمی رو با متن جدید عوض کن
}
```

### قدم 6️⃣: ذخیره کن
بعد از ذخیره، سایت **خودکار** به‌روز می‌شه! 🚀

---

## راه سریع‌تر (برای کاربران حرفه‌ای) 💪

مستقیماً فایل `fa.json` رو باز کن و ویرایش کن:
```
📁 src/i18n/locales/fa.json
```

فایل به بخش‌های مختلف تقسیم شده با کامنت‌های واضح:
```json
{
  "_comment_nav": ">>> متن‌های منوی ناوبری <<<",
  "nav": {
    "home": "خانه",  ← اینجا ویرایش کن
    "about": "درباره من"
  },
  
  "_comment_hero": ">>> بخش صفحه اصلی <<<",
  "hero": {
    "greeting": "غزال حسینی",  ← اینجا ویرایش کن
    ...
  }
}
```

---

## 🔍 پیدا کردن متن در JSON

### نحوه خواندن آدرس:
```
hero → greeting
```
یعنی برو به بخش `hero` و کلید `greeting` رو پیدا کن:
```json
"hero": {
  "greeting": "غزال حسینی"  ← اینجاست!
}
```

### مثال دیگه:
```
contact → form → name
```
یعنی:
```json
"contact": {
  "form": {
    "name": "نام شما"  ← اینجاست!
  }
}
```

## ⚠️ نکات مهم

### ✅ انجام بده:
- فقط متن داخل گیومه رو تغییر بده
- از UTF-8 برای ذخیره استفاده کن
- بعد از هر خط (غیر از آخرین خط) ویرگول بذار

### ❌ انجام نده:
- کلیدها (keys) رو تغییر نده
- گیومه‌ها رو فراموش نکن
- ویرگول‌ها رو حذف نکن

## 🎨 مثال عملی

### می‌خوام متن دکمه "مشاهده نمونه کارها" رو عوض کنم

#### قدم 1: پیدا کردن
در `FARSI_TEXTS.md`:
```
دکمه 1: مشاهده نمونه کارها  ← این متن رو می‌خوام عوض کنم
```

#### قدم 2: پیدا کردن در JSON
در `fa.json`:
```json
"hero": {
  "viewWork": "مشاهده نمونه کارها"  ← اینجا باید تغییر بده
}
```

#### قدم 3: ویرایش
```json
"hero": {
  "viewWork": "دیدن پروژه‌ها"  ← متن جدید
}
```

#### قدم 4: ذخیره
`Ctrl + S` یا `File → Save`

#### قدم 5: چک کردن
سایت خودکار به‌روز می‌شه! 🎉

## 💡 سوالات متداول

### Q: اگه اشتباه وارد کردم چی می‌شه؟
A: فایل JSON خطا می‌ده. یه JSON Validator استفاده کن یا متن رو برگردون به حالت قبل.

### Q: چطوری بفهمم درست شده؟
A: سایت رو در مرورگر چک کن. اگه متن تغییر کرد، درسته!

### Q: چند تا فایل باید ویرایش کنم؟
A: فقط یکی: `fa.json` برای فارسی، `en.json` برای انگلیسی

### Q: هات ریلود (Hot Reload) چیه؟
A: بعد از ذخیره، سایت خودکار به‌روز می‌شه بدون نیاز به رفرش دستی!
