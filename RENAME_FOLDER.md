# راهنمای تغییر نام فولدر assests به assets

## مشکل
فولدر `assests` نام اشتباهی دارد و باید به `assets` تغییر کند.

## راه حل

### مرحله 1: متوقف کردن سرور
در ترمینال که `npm run dev` در حال اجراست، `Ctrl + C` را فشار دهید تا سرور متوقف شود.

### مرحله 2: تغییر نام فولدر
در PowerShell دستور زیر را اجرا کنید:

```powershell
Rename-Item -Path "assests" -NewName "assets"
```

یا در File Explorer فولدر `assests` را به `assets` تغییر نام دهید.

### مرحله 3: اجرای مجدد سرور
```bash
npm run dev
```

## نکته مهم
فولدر در حال حاضر توسط سرور Vite استفاده می‌شود، بنابراین ابتدا باید سرور متوقف شود.
